Using linux in the home directory

1) unpack the file bibliotheque-graphique-iut-1.1.tar.gz 

2) Open terminal (Ctrl+Alt+T) and run the commands:
	
	$ cd bibliotheque-graphique-iut-1.1/

	$ ./configure --prefix=/usr --disable-exemples

	$ make

	$ make install

The library should be installed
